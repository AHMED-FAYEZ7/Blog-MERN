# Blog-MERN

## Live Demo: 



## Cloning repo
    $ git clone https://github.com/HosamAboabla/Blog-MERN.git
    $ cd Blog-MERN

## Installing dependencies
    $ npm install
    $ npm run client-install
    $ npm run client-build

## Running command
    $ npm run start

## http://localhost:5000/
